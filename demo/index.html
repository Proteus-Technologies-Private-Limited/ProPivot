ProPivot interactive demo — engine + virtualized grid (v)

Try: drag fields between Field-List zones (incl. Filters — the bar below gets a member picker). Hover a measure column header and click the ⚙ button to change that measure's aggregation in a modal (each measure is independent). Switch Layout → Flat to show each row field as its own column (the pivot is preserved). Click a measure header to sort rows by it (▼/▲); in Flat layout click a row-field header to sort its members. Double-click any value cell to drill through to the underlying rows. The toolbar above the grid has Fields / Export / Fullscreen. Conditional formatting highlights large/small sales; calculated measure Avg Price = sum('sales')/sum('qty') is always shown.
